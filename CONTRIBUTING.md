# 贡献指南

感谢你考虑为本项目做出贡献！

---

## 🤝 如何贡献

### 报告问题

如果你发现了错误或有改进建议：

1. 在 [Issues](https://github.com/yourname/awesome-clash-vpn/issues) 页面搜索，确认问题未被报告
2. 创建新 Issue，包含：
   - 清晰的标题
   - 详细的描述
   - 截图或代码（如适用）

### 提交资源

如果你想添加新资源：

1. Fork 本仓库
2. 创建分支：`git checkout -b feature/add-resource`
3. 编辑 README.md，在合适位置添加资源
4. 提交：`git commit -m 'Add: 新资源名称'`
5. 推送：`git push origin feature/add-resource`
6. 创建 Pull Request

---

## 📝 贡献规范

### 资源添加规范

- ✅ 确保资源真实可用
- ✅ 添加简短描述和推荐理由
- ✅ 放在合适的分类下
- ✅ 保持格式一致
- ❌ 不要添加低质量或失效资源

### 格式要求

```markdown
| 名称 | 特点 | 链接 |
|------|------|------|
| 资源名 | 简短描述 | [链接](url) |
```

### Commit 规范

使用语义化提交：

- `Add: 添加新资源`
- `Update: 更新资源信息`
- `Fix: 修复错误`
- `Remove: 移除失效资源`
